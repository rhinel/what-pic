# what-pic
图什么，网站前台<br>
Built with Vue 2.0 + vue-router + vuex, with server-side rendering.

## Build Setup

**Requires Node.js 7+**

``` bash
# install dependencies
npm install

# serve in dev mode, with hot reload at localhost:8080
npm run dev

# build for production
npm run build

# serve in production mode
npm start
```
